Common questions

How does "no repeats" work?

When enabled, the app excludes any student who's already been called this round. After everyone's been picked once, the round resets automatically — or tap "Reset round" to start over manually.

Multiple class periods?

v1 has one shared roster (your active class). Multiple-class support is on the v1.1 list — let us know if it's blocking your daily use.

Can I import a CSV / paste a class list?

v1 takes one name at a time. CSV import is on the roadmap.

Group sort with uneven groups?

The app distributes students round-robin into K groups. If you have 23 students and 4 groups, you'll get 6/6/6/5. Some groups will be one student smaller.

Apple Watch / parent-pairing / iPad?

iPad supported. Apple Watch and other variants on the roadmap.
